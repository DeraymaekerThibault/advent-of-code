# Advent of Code

Personal solutions to [Advent of Code](https://adventofcode.com) puzzles — written for learning and fun.

## Languages

Solutions are written in both **Rust** and **Python**, sometimes solving the same puzzle in both to compare approaches.

## Structure

```
{year}/
  {day}/
    src/main.rs   # Rust solution
    code.py       # Python solution
    input.txt     # Puzzle input
```

## Years

| Year          | Days Solved |
| ------------- | ----------- |
| [2025](2025/) | 1 – 7       |

## Running Solutions

**Rust** (from the year's workspace):

```sh
cd 2025
cargo run -p day01
```

**Python**:

```sh
python 2025/01/code.py
```

## Goals

- Practice Rust — ownership, iterators, performance
- Keep Python solutions clean and idiomatic
- Have fun with the puzzles :)
